# WRN Cleaner
All-In-One Batch Script PC Maintenance and Cleaner for Windows PC

## Basic Features
* Defragment
* DiskClean
* Windows Error Log Cleaner
* Recycle Bin Cleaner
* Clean Temp Folder
* Clean Prefetch Folder
* System File Checker
* Auto Shutdown and Reboot When Done
